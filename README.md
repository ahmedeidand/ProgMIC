# ProgMIC

Web app that types text into your system using xdotool.

## Requirements

- PHP
- xdotool: `sudo apt install xdotool`
- Web server

## Setup

Clone repo, install deps, run server (e.g., `php -S localhost:8000`)

## Usage

Open index.html, enter text, submit. Types the text via xdotool.

## Note

Requires X11 for xdotool.
